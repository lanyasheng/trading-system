## 重要判断记录

### 2026-02-24 周期股异动事件
**背景**: 自选股监控首次检测到多个周期股（资源+石油+新能源ETF）同步异动，而消费股（茅台-1.3%）资金流出，呈现明显的板块切换特征。

**异动标的**:
- 兴业银锡 +6.84%（早盘放量）
- 中国石油 +4.93%
- 金风科技 +4.74%
- 紫金矿业 +4.13%
- 相关ETF涨幅1.55%-2.88%

**技术特征**:
- 所有异动股均为早盘9:30-10:00放量，主力抢筹明显
- 成交量显著高于平日（紫金矿业单日84.69亿成交额）
- 多个标的突破关键压力位

**初步判断**:
资金从消费股（白酒）轮动至周期资源板块，可能契机包括：商品价格上涨预期、能源安全主题、一带一路相关催化剂。

**后续跟踪要点**:
1. 异动持续性（是否一日游）
2. 相关ETF是否出现溢价或封板
3. 是否带动其他周期股（煤炭、钢铁、有色）
4. 消费股是否继续资金流出

**监控阈值调整建议**:
当前自选股监控阈值可降低至单日涨跌幅>1%（原为3%），以更敏感捕获周期股波动。
