## 晨报模板（<=2000字）

```
🕗 A股晨报 | YYYY-MM-DD（周X）

📌 隔夜要闻（3-5条）
• 要闻 - 影响分析

📊 全球市场
• 美股: 三大指数涨跌幅
• 商品: 黄金/原油/铜
• 汇率: 美元指数/人民币

🎯 今日关注
• 经济数据/政策/个股事件

📈 自选股盘前（全量展示）
• 代码 名称: 昨收价 涨跌幅

⚠️ 风险提示（1-2条）
```

## 盘前策略模板（<=1500字）

```
🕘 盘前策略 | HH:MM

📊 技术信号摘要
• 多头/空头/中性标的分类
• 强信号(>=5分)标的重点展开

• 重点标的主力净流入/净流出
• 资金方向结论

🎯 今日策略

• 风险控制建议
```

## 收盘复盘模板（<=2000字，7模块）

```
📊 完整盘后复盘 | YYYY-MM-DD（周X）

1️⃣ 大盘总览
• 上证/深成/创业板涨跌幅 + 成交额变化
• 一句话市场状态

2️⃣ 宏观/政策/海外驱动（3-5条，每条写影响）

3️⃣ 板块轮动与风格
• 领涨/领跌 + 风格切换结论

4️⃣ 重点池全展开（不省略）
• 每个: 收盘价 + 涨跌幅，缺失标注"数据缺失"

5️⃣ 资金面
• 重点标的主力净流入/净流出
• 北向资金（异常时标注+替代口径）

6️⃣ 基本面/估值
• 股票: FV_real 优先
• ETF: FV_proxy

7️⃣ 综合结论 + 次日执行计划
• 结论3条 + 可执行动作3条（含风控）
```


## 商品/美股异动模板（<=1200字）

```
🔔 市场异动 | HH:MM

1) 风险偏好一句话
2) 异动清单（超阈值品种: 价格+涨跌幅）
3) 关键驱动（2-3条）
4) 对A股影响
5) 风险提示与执行建议
```
